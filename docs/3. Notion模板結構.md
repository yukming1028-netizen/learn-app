# 📚 Notion 模板結構（專案管理與知識庫）

> ✅ **用途**：作為團隊協作中心，整合需求、任務、進度、文件與資料庫。

---

## 🔧 建議建立以下頁面與資料庫

### ➤ 主頁：`親子學伴專案總覽`
- 標題 + 專案簡介  
- 進度條（完成度）  
- 快速連結：  
  - 📄 [需求文檔](#)  
  - ✏️ [Figma 設計稿](#)  
  - 📊 線上原型測試回饋  
  - 💡 每週會議紀錄

---

### ➤ 數據庫 1：`功能需求 (FR)`
| 欄位 | 說明 |
|------|------|
| 名稱 | FR-01, FR-02... |
| 描述 | 如「子女端無需註冊，掃碼綁定」 |
| 優先級 | P0 / P1 / P2（下拉） |
| 狀態 | 待辦 / 進行中 / 已完成 |
| 所屬模組 | 認證 / Dashboard / 學習引擎 |
| 相關設計稿 | 連結至 Figma Frame |

---

### ➤ 數據庫 2：`任務清單 (Todo)`
- 按 Sprint 分組（如 Sprint 1: 綁定流程）  
- 每個任務包含：  
  - 責任人  
  - 截止日  
  - 附件（設計稿、API 文件）  
  - 完成標準（Checklist）

---

### ➤ 數據庫 3：`子女資料庫`
| 欄位 | 說明 |
|------|------|
| 子女姓名 | 張小明 |
| 年級 | 小二 |
| 綁定家長 | 李媽媽（連結至使用者資料庫） |
| 每日目標時長 | 20分鐘 |
| 學科偏好 | 數學 > 英語 > 語文 |
| 最近活動 | 2026-05-21 17:30 完成數學5題 |
| 累計學習時數 | 4h 18m |
| 已解鎖貼紙 | 🐶小熊、🌟閃電 |

> ✅ 可建立「看板視圖」：按「進度」分組，便於追蹤每個孩子的狀態。

---

### ➤ 數據庫 4：`題庫管理`
| 欄位 | 說明 |
|------|------|
| 題目 | “5 + 8 = ?” |
| 學科 | 數學 |
| 年級 | 小一 |
| 難度 | 簡單 |
| 正確率（全體） | 92% |
| 做題平均時間 | 12秒 |
| 标籤 | 加法、個位數、進位 |
| 類型 | 選擇題 / 輸入答案 / 拖曳匹配 |

> ✅ 可導出為 JSON，供 App 端離線載入。

---

### ➤ 數據庫 5：`用戶資料庫`
- 家長帳號（郵箱、註冊時間、綁定子女清單）  
- 子女唯一識別碼（UUID）
