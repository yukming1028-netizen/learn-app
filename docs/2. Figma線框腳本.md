# 🖥️ Figma 線框圖腳本（UI 設計指引）

> ✅ **用途**：提供 UI/UX 設計師明確的畫面結構與互動邏輯，無需猜測。  
> ✅ **建議建立 3 個 Frame 組**：`Parent Dashboard`、`Child App Home`、`QR Bind Flow`

---

## 🎯 1. 家長端主畫面 — Dashboard（Web / iOS）

### 🔹 Frame 名稱：`Parent Dashboard - Overview`
- **頂部導航**：  
  - Logo + "親子學伴"  
  - 右上角：頭像 → 下拉 [設定、登出]
- **子女卡片區**（最多 3 個）  
  > 每張卡包含：
  > - 子女名字 + 年級（如「小明 · 小二」）  
  > - 圓形進度條：今日完成率（如 85%）  
  > - 總學習時數：`2h 14m`  
  > - 最近一次活動：`30分鐘前完成數學題`  
  > - 按鈕：「查看詳情」→ 跳轉至該子女專頁
- **統計圖表區**（水平滾動）：  
  - 📊 **7日趨勢圖**：學習時間 vs 正確率（折線圖，雙Y軸）  
  - 📚 **學科分布**：餅圖（數學40%、語文30%、英語25%、科學5%）
- **計劃區**：  
  - 標題：“本周學習計劃”  
  - 狀態標籤：✅ 已啟用 / ⏸️ 暫停  
  - 內容：「週一至五 17:00–18:00｜數學3題 + 語文2詞」  
  - 按鈕：“編輯計劃” → 開啟計畫編輯器

> 💡 **設計規範**：  
> - 主色調：暖橙 (#FF6B35)、柔藍 (#4A90E2)、米白 (#F8F5F0)  
> - 字體：思源黑體 Medium（中文） + Inter（英文）  
> - 圖示：使用 Material Symbols 或 FontAwesome 簡潔線性圖標

## 🎮 2. 子女端主畫面 — Home Screen（Android / iPad）

### 🔹 Frame 名稱：`Child App - Home`
- **背景**：卡通風雲朵與星星動態背景（輕微飄浮）
- **中央大按鈕**：  
  > “今天要學習嗎？” → 點擊進入「今日任務」
- **下方功能區**（水平滾動 icon）：  
  - 📚 **今日任務**：藍色圖示 + 文字，顯示剩餘題數（如“3/5”）  
  - 🎁 **我的收藏**：已解鎖的貼紙與寵物（點擊可查看）  
  - 🌟 **挑戰模式**：額外自由題庫入口（灰色標註「家長未設定」）  
  - 💬 **給爸爸媽媽的話**：語音留言按鈕（錄製 ≤10秒）
- **底部導航條**：  
  - 主頁（亮色）｜我的寵物｜錯題本｜設置

> 🎮 **交互細節**：  
> - 每答對一題 → 小動物跳起來 + 動畫星星飛舞 + 音效「叮！」  
> - 錯誤時 → 小熊搖頭 + 輕柔鼓勵語音：“再試一次，你一定可以！”  
> - 所有文字使用**大字體、高對比度**（適合兒童閱讀）

## 🔗 3. 綁定流程 — QR 掃碼與激活

### 🔹 Frame 名稱：`QR Bind Flow`
1. **家長端** → 點擊「綁定子女」  
   - 弾出全螢幕動態 QR Code（含加密 UUID + 時效 5 分鐘）  
   - 說明文字：“請讓孩子用 App 掃描這個碼”

2. **子女端** → 首次啟動自動進入「掃碼模式」  
   - 全螢幕相機視窗，中央有對焦框  
   - 成功掃到後：  
     - 播放音效 + 3D 粉紅色星星爆發動畫  
     - 文字：“哇！你被媽媽綁定了～”  
     - 出現子女名字輸入欄（預設為「小寶貝」，可修改）  
     - 點擊「完成」→ 自動同步至家長端

3. **家長端收到通知**：  
   > “恭喜！小寶貝已成功綁定。現在可以設定第一個學習計劃了。”

> ✅ 安全提醒：QR 碼每 5 分鐘自動失效，無需密碼、無需帳號輸入。
